Algonive-Sentiment-Analysis-and-Actionable-Complaint-Extraction-from-App-Store-Reviews
________________________________________________________________________________________________________________________________________________________

🔍 Overview

A data science project that analyzes app store reviews to detect user sentiment and identify common complaints using NLP techniques.
This project analyzes app store reviews to identify user complaints automatically.

_____________________________________________________________________________________________________________________________________________________

⚙️ Technologies Used

- Python  
- Pandas (Data handling)  
- NLTK (Text preprocessing)  
- TextBlob (Sentiment analysis)  
- Scikit-learn (Bigram analysis)  
- Matplotlib (Visualization)  
- WordCloud (Visualization)  
- Google Play Scraper (Data collection)

______________________________________________________________________________________________________________________________________________________

🚀 Methodology

- Collected reviews using Google Play Scraper  
- Cleaned text (removed noise, stopwords)  
- Performed sentiment analysis (Positive, Neutral, Negative)  
- Extracted common phrases using bigram analysis  
- Visualized results using charts and word cloud  

_______________________________________________________________________________________________________________________________________________________

✨ Features

- Data collection using Google Play Scraper
- Text preprocessing (cleaning, tokenization)
- Sentiment analysis (Positive, Neutral, Negative)
- Bigram analysis for common issues
- Visualization (graphs and word cloud)
